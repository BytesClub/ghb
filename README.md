ght is a command line tool to fetch and update GitHub Issue and Pull Request from terminal

### Installation

* Using _**npm**_
```bash
$ npm install ghb -g
```

### Usage
* Configure _**ghb**_ with local repo
```bash
$ ghb init <github-url>
```
* Get latest issues/pr from GitHub
```bash
$ ghb issues
$ ghb pulls
```
